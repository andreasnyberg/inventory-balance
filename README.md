# inventory-balance

Inventory balance CLI

## Installation
* Clone repo
* `npm install`
* `npm start`

## Usage/commands

Display current inventory balance:
```
L
```
---
Add X products to inventory.
```
I + <amount>
```

*For example:* `I13` adds 13 products.

---
Sell X products from inventory.
```
S + <amount>
```

*For example:* `S11` sells 11 products.
