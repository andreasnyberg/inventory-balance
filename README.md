# inventory-balance

Inventory balance CLI

## Installation
* Clone repo
* `npm install`
* `npm start`

## Usage/commands

Display current inventory balance:
```
L
```
---
Add X products to inventory.
```
I + <amount>
```
---
Sell X products from inventory.
```
S + <amount>
```
---

*Examples:*  
`I13` adds 13 products.  
`S11` sells 11 products.
